Understanding the Map
The map displays different points, each representing a user's location.
Each point has a different color, representing the status of the user:
  • Green: The user is currently available.
  • Orange: The user's status is under review.
  • Red: The user is currently unavailable.
By clicking on a point, or the point button you can go to the profile of that user see what they have available and open a chat with them.
The '+' button at the bottom right corner of the page allows you to add a new point of your own.
